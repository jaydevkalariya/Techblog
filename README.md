# Techblog
Related to techical blogs..

How to run project?
Run index.jsp file.
see response on localhost/9494/Techblog

Functionalities:

1) Admin side:
  -add post
    i) special
    ii) normal
    
  -delete post
  -block user
  -like & comment
  -Add categories for post
  -Add subscription code
  
2) User side
  -add post
  -read blogs
  -like & comment
  -delete own posts
  -search posts
  
3) profile management using session
